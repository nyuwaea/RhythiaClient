##### USE CAUTION WHEN MANUALLY EDITING SKIN FILES #####

All skinning should be done through the in-game editor! Only edit skins through files if you know what you are doing!

Certain skins such as built-in or online skins are not meant to be edited directly. If you would like to edit such a skin, use the Copy feature in the skin editor to duplicate it first!
If there are features or tricks related to skinning only available through manual editing which you would like official support for, please let us know through GitHub or our Discord server.
